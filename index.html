Description:

There are terrorists trying to bomb the city, you are the only soldier
in city so it is your job to stop them. Defeat all terrorists before they kill you.
Be alert, terrorists spawn anywhwere and they shoot you.
You have only 3 lives, everytime they shoot you, your life decreases.
You have a limited ammo of 12 bullets.
The game will be over if you exhaust your ammo or use all lives.
The only way to win is by defeating all terrorists.

Instructions:

Press 'S' to start the game.
After the game finishes press 'R' to restart.
Use the up, down, right and left arrow keys to move in that particular direction.
Press the spacebar to shoot one bullet at a time.

Libraries used:

p5.js and p5.play.js

Libraries are files where developers collect pre-written code and
functions which can later be used by a programmer.

Tools used:

All objects in the game are made using sprites.
Sprites are used where you need to add images or animations to objects.
They don't obey the laws of physics.

Explaining the code:

83 is the value given to 'S', so when 'S' key is pressed, the game will go to gameState 1 which is the play state, and a sound is played.
In play state a background image is added and enemyShoot and spawnEnemy functions are called.
The if conditions contain movements for the Soldier.
spawnEnemy creates a terrorist sprite, spawns them randomly and adds images randomly.

Here bullets for the terrorist are created.
enemyShoot is the function which makes the terrorist automatically shoot us.
Because there are multiple bullets, they are added in to group.

Here bullets for the Soldier are created.

32 is the value for the spacebar.
spawnBullets is called when spacebar is pressed, ammo is decremented and a sound is played.
Bullets are only shot when game is in play state (gameState=1).

The game only ends when any one of these conditions is true.
The conditons are:
life=0 (meaning all lives are used)
ammo=0 (meaning all bullets are used)
kills=10 (meaning all terrorists have been defeated)
